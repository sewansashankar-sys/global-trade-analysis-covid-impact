# Global Trade Analysis: Before and After COVID-19

## Project Overview

This project analyzes international trade patterns before and after the COVID-19 pandemic using trade datasets from 2016, 2018, and 2023.

The objective is to understand how global trade dynamics changed due to disruptions caused by COVID-19 and identify recovery trends across countries and regions.

## Objectives

- Compare trade performance before and after COVID-19
- Analyze country-wise trade trends
- Study changes in logistics performance
- Identify top-performing and declining economies
- Visualize global trade network changes

## Dataset

Data Sources:
- World Bank
- WTO Trade Statistics
- Logistics Performance Index (LPI)

Years Analyzed:
- 2016
- 2018
- 2023

## Methodology

1. Data Collection
2. Data Cleaning
3. Data Integration
4. Exploratory Data Analysis
5. Visualization
6. Comparative Analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NetworkX
- Jupyter Notebook

## Key Findings

- Trade volumes declined significantly during the pandemic period.
- Recovery trends became visible by 2023.
- Countries with stronger logistics infrastructure recovered faster.
- Trade networks became more concentrated during disruption periods.
